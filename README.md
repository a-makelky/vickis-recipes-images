# Vicki Makelky's Recipe Collection - Image Archive

This repository contains scanned images of handwritten recipe cards from Vicki Makelky's collection.

## Organization

Images are organized by recipe name for easy reference.

## Usage

These images are linked from the Notion recipe database at:
[Vicki Makelky Recipe Repo](https://www.notion.so/30f2b47bfc6980f39bebfa715ade2e0b)

## Adding New Images

1. Place image in root folder with descriptive name: `recipe-name.jpg`
2. Commit and push to GitHub
3. Update Notion recipe page with image URL:
   `https://raw.githubusercontent.com/a-makelky/vickis-recipes-images/main/recipe-name.jpg`
